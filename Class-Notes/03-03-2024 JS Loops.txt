
1. for loop: i as an index or counter

SYNTAX:

for(initialization; termination_condition; update) {
    // block
}




ITERATION: Each time your loop executes
INFINITE LOOP: This is a loop having a logical mistake and does not stop (goes forever)



-------------------------------------------------------------------------------------------------------------------



2. while LOOP

SYNTAX:

while(condition) {
    //block
}



-------------------------------------------------------------------------------------------------------------------



3. do-while

- while loop checks the condition and if it is true, it executes the code
- do-while loop executes the code block first, then checks the condition
- Beceause of the difference above, do-while will execute the code-block at least once all the time

SYNTAX:
do {

} while(condition);



-------------------------------------------------------------------------------------------------------------------



LOOP CONTROL STATEMENTS:
break - is used to terminate the loop
continue - used to skip the current iteration
